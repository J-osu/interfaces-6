# StreamDeck UI Wiki - Home

Bienvenido a la Wiki oficial de **StreamDeck UI**.

Este proyecto es una interfaz gráfica moderna inspirada en el software de Elgato Stream Deck, construida como una aplicación web Reactiva, robusta y atractiva. 

## Índice de la Wiki

1. **[System Check & Hardware Detection](./SystemCheck.md)**
   Conoce cómo la aplicación evalúa las capacidades técnicas de tu máquina nativamente antes de entrar.
2. **[Dashboard & Swapy Drag and Drop](./Dashboard.md)**
   Aprende cómo está estructurado el tablero principal de widgets interactivos y cómo logramos el drag and drop reactivo.
3. **[Testing Backend y Frontend](./Testing.md)**
   Detalles de las pruebas unitarias y mocks instalados para asegurar la fiabilidad de la lógica de la aplicación.

## Stack Tecnológico

- **Vue 3 (Composition API)**
- **TypeScript** para un código mantenible y libre de errores en compilación.
- **Vuetify 3** para componentes de diseño y maquetación de Layouts.
- **Pinia** para la gestión reactiva global del estado.
- **Swapy** para los reordenamientos Drag & Drop nativos.
- **Vite** como empaquetador veloz.
- **Vitest & JsDOM** para la capa de tests unitarios.
